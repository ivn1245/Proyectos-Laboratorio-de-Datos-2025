El objetivo de este proyecto fue hacer un análisis de la distribución demografica de colegios y establecimientos educativos en Argentina. 
Comenzando con la construcción de la base de datos, se consiguieron las tablas del portal nacional de datos públicos, se diseñó un diagrama entidad relación y un modelo relacional. Luego se limpiaron las tablas originales y se crearon las tablas modelo.
Una vez que ya se contaba con la base de datos armada, se utilizaron consultas SQL para poder realizar el análisis.
Todo el proceso y las conclusiones quedaron documentadas en el informe, mientras que la limpieza de las tablas originales con Pandas, las consultas SQL y los gráficos se pueden encontrar en el archivo "TP1.py".
